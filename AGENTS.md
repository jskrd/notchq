# AGENTS.md

## Setup commands

- Install deps: `pnpm install`
- Run format: `pnpm format`
- Run knip: `pnpm knip`
- Run lint: `pnpm lint`
- Run tests: `pnpm test`
- Run type check: `pnpm typecheck`

## Code style

- TypeScript strict mode
- Prettier default formatting
- Use functional patterns where possible
